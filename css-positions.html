    ¿Que es el Posicionamiento en CSS?

    Como vimos en la pagina anterior la propiedad float nos permite modificar la posicion de una caja, con diferentes opciones, entre ellas la de arriba abajo o derecha izquierda, y tambien nos permite editar el orden del documento html, la propiedad POSITION nos permite hacer algo parecido. Esta propiedad funciona de manera parecida a como se comportan los margenes

    Al igual que en los anteriores esta propiedad tiene algunos tipos
    • static (default) - Es el como dejar la caja o el elemento como esta sin cambiar su orden en el documento html
    • relative
    • absolute
    • fixed -
    • sticky -
    Tenemos las propiedades de siempre: top, bottom, left, right. Pero se le suma index (Las propiedades top y left van a tener prioridad sobre bottom y right)
    • index - Lo que representa es la profundidad cual puede ir mas adelante cuando dos elementos se sobreponen, todo eso lo iremos viendo.

    Posicionamiento Static

    Este es el valor por defecto, este respeta el flujo o el orden por el que va css, no cambia de posicion, entonces basicamente, no se puede posicionar, si miran el siguiente ejemplo notarán que aunque le he aplicado estilos y he intentando cambiarle de posicion la propiedad static bloquea todos estos estilos.

    Posicionamiento absolute

    El elemento pierde sus dimensiones y posicion original en el flujo del documento. Si se mueve toma como referencia 2 elementos:
    1) El primer ancestro con posicionamiento relativo
    2) Si no encuentra un ancestro relativo, se mueve con respecto del documento HTML

    Posicionamiento fixed

    El elemento pierde sus dimensiones y posicion original en el documento HTML, asi que no sigue esa linea por que se queda fijo, cuanto se mueva el documento hacia arriba o hacia abajo, lo que pasa es que sigue pegado.

    Posicionamiento sticky

    Sticky es una combinacion de poisicionamiento relative y fixed, las propiedades top,left,bottom,right no mueven estos elementos, además no puede ser hijo de otras etiquetas, para que funcione debe ser hijo directo del body. Ademas el centrado de una caja es normal, basado en margin.

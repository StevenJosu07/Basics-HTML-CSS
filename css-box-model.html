Modelo de Caja

Para html y css todo el contenido y cada etiqueta es como una caja y consta de:

  • El contenido (content) que es lo que se ve en html por lo general texto
  • El borde (border) lo que delimita el contenido (Puedes poner un border solid para ver cual es)
  • El relleno (padding) las distancias internas o lo que hay entre el contenido y el borde
  • El margen (margin) es la distancia externa entre la caja y otras cajas

En cuanto al padding y margin hay un pequeño atajo y es que por lo general deben llevar un valor que es el que les hace actuar por ejemplo ponerle 10 pixeles, si en nuestra hoja de estilos a ese valor se le agrega otro al lado se le aplicara a los lados, en resumen al primer valor se le aplicara arriba y abajo y el segundo valor lo aplicara a la derecha e izquierda y asi seguira entre mas valores pongan todos como la manecilla del reloj, valor 1, 2, 3, 4. Las cajas tienen cuatro lados:

  • Arriba (Top)
  • Derecha (Rigth)
  • Abajo (Bottom)
  • izquierda (Left))

Estos cuatro lados sirven para especificar a que parte de la caja quieres ponerle un estilo. Ademas hablando de especificar se puede editar este caja con (width & Heigt) añadiendole sus respectivos tamaños. Tengo que recordar la importancia de distinguir las etiquetas:

  • Border: Este es para representar que haya un borde, siempre debes tener este borde si quieres añadir un estilo lo mejor es ponerlo en border-style
  • Border-width: Este es para ver como de ancho quieres el borde lo puedes medir por pixeles o por las medidas predeterminadas, thin, medium, tick.
  • Border-color: Con este le puedes dar color al borde y puede ser lo que estan nombrados ahi o con el codigo de color

A las ultimas dos debemos recordar que tienen aun mas estilos por que es mas especifico, ya que cada lado es distinto, lo que hace es juntar esos lados.

Cajas de bloque vs caja de linea

Cajas de línea
  • Ocupan solo el espacio necesario para mostrar el contenido
  • No tienen dimensiones modificables, se quedan como estan
  • Permiten otros elementos a su lado

    Propiedad display

    Esta propiedad que le dice al navegador como se va a visualizar un elemento html a css, es muy extensa esa propiedad ya que va desde eliminar el contenido a ponerlo en linea en bloque o de diferentes formas:

    • inline - obliga a la caja a ser caja de linea
    • block - obliga a la caja a ser caja de bloque
    • inline-block - Se comporta como un elemento de línea pero acepta modificar sus dimensiones (alto y ancho)
    • none - se oculta la caja pero se carga el contenido
    • table . se usa unicamente para las tablas (se agrega por defecto)
    • list-item - esta es la propiedad que se le da las lista (li) (Se agrega por defecto)
    • flex - ve mi curso de Flexbox CSS
    • grid - ve mi curso de Grid CSS

    Propiedad Float & Clear

    Float

    La propiedad float nos va permitir cambiar la posición horizontal de las cajas, nos va permitir llevarlas a la izquierda o a la derecha su valor por defecto es float none, pero podemos editarla para que flote a la izquierda con float:left o a la derecha con float:right. ¿Que es lo que hacen exactamente? Buscar el lugar mas cercano a la derecha o a la izquierda, pero pierden el flujo del documento html, se les quita su posición natural, pierden el orden que se les da.
    Asi que tenemos un problema con el orden que le queremos dar las posiciones de html son como una fila de banco, cada contenido es como una persona, si una de esas personas sale de la fila de la derecha va a haber un problema pues estan intentando ocupar los dos una misma fila, obviamente esto no es un prolema si queremos hacerlo a posta, pero no es la mejor manera cuando necesitamos manejarlo de otra manera. Es por eso que en el segundo ejemplo se le quita el color de fondo, por que aunque estan en el mismo div el contenido se sale del div por el css, entonces esquivan el color de fondo, asi que es como si la div no tuviera nada dentro.

    Centrado de cajas

    Si queremos centrar una caja como se ve en diferentes sitios web en donde dejas espacios a la izquierda y derecha para poner anuncios, lo que debemos es usar la propiedad margin para que mueva la caja en el exterior, es importante no confundirse con la propiedad text aling center, que centra el texto de la caja pero no la caja en sí.
    La propiedad es "margin-left:auto & margin-right:auto" se utilizan ambas. Lo que hacer es empujar la caja a los respectivos lados, pero como utilizamos left & ritgh al mismo tiempo lo que hacen es chocar y quedarse en el centro,
